This is really a simple game to build, and was my first real jump into Javascript to try to get to learn the language and HTML Canvas at the same time. this early version is still pretty crude, but it shows off using bit operation to generate all the shapes so they can be stored as simple integers. 

            
            const LAYOUTS = [
                [51],                   //square
                [240, 8738],            //line
                [99, 306],              //s-left
                [54, 561],              //s-right
                [116, 1570, 368, 547],  //L
                [1604, 1136, 550, 1808],//backwards L
                [114, 610, 624, 562]    //T
            ];

You can try out the number generator here and draw your own shapes:


Every tile has a bit equivelent, starting with top row with bits: 0 to 3 and the second row 4 to 7, third 8 to 11, and finally the forth row 12 to 15. when the bit is turned on that value is 'or'ed with any other squares to get a final value.

The game is broken up into 10 squares wide by 20 squares high; each square at 10 pixels in height and width. when the timer advances a tick; the game will take the current shape of 4 tiles and advance one down while checking for collisions. if there is a collision it stops the shape, copies the 'on' tiles to the back buffer and then checks for completed row
